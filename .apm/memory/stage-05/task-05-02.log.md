## Details
1. Antes de escrever, validei o estado real das telas em vez de descrever apenas o que constava no planejamento (conforme "Instruction Accuracy"): confirmei via leitura de código os campos exatos do formulário de cadastro de ave (`app/plantel/novo/NovoCadastroForm.tsx`, `lib/aves/actions.ts`), o comportamento da Ficha da Ave (`app/plantel/[id]/FichaAve.tsx`), os filtros da Lista do Plantel (`app/plantel/PlantelList.tsx`), a lógica exata das duas travas de ninhada (`lib/aves/compatibility.ts`, `NovaNinhadaForm.tsx`, `/api/parentesco`), o cálculo de taxa de eclosão (`lib/ninhadas/service.ts`), e a navegação por clique da árvore genealógica (`app/arvore/[id]/ArvoreView.tsx`).
2. Confirmei que a Trava 1 (compatibilidade de reprodutores) é enforced por **filtragem dos próprios campos de seleção** (apenas aves compatíveis aparecem no select), não por uma mensagem de erro pós-seleção — documentado assim no guia para evitar confundir o usuário à procura de uma mensagem de erro que não existe.
