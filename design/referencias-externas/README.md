# Referências externas (não fazem parte do design aprovado)

Materiais de referência trazidos pelo usuário a partir de uma ferramenta concorrente
(MyBirds — meuspassaros.com.br), para eventual consulta futura. **Não são fonte de
verdade visual do projeto** — a pasta `design/` (mockups `.dc.html` e o Design System)
continua sendo a única referência oficial de layout, cores e componentes, conforme
regra do projeto. Qualquer incorporação de elementos daqui exige decisão explícita e
atualização do design aprovado antes de qualquer trabalho de interface.

## Arquivos

- `carteira-mybirds-exemplo.pdf` — exemplo de "carteira" (cartão de identificação
  compacto com foto/dados da ave + árvore genealógica resumida ao lado), formato de
  cartão menor que o certificado de pedigree completo.

## Observações (2026-07-05)

O usuário também compartilhou, na mesma conversa, uma imagem de um "Certificado"
maior da mesma ferramenta (não salva em arquivo — veio colada na conversa, sem
caminho de disco). Esse certificado mostra:

- Árvore genealógica de **5 gerações** (o Spec do Ninhal define 3 gerações).
- Selo/logo do criatório, QR code, dados de contato e CTF do responsável.
- Coloração por sexo (azul/rosa) nas caixas da árvore.

Nenhuma decisão de escopo foi tomada a partir desses materiais — ficam aqui apenas
como referência, caso o usuário queira discutir extensões futuras (ex: um formato de
"carteira" compacta adicional ao pedigree, ou aumentar a profundidade da árvore).
